Ficha Metodológica

Nombre del Indicador:

Tasa de homicidios por cada 100.000 habitantes

Definición:

La Tasa de homicidios por cada 100.000 habitantes es la relación del total de muertes por Homicidios por cada 100.000 habitantes.

Periodo:

2013 - 2017

Fuente:

Base de datos de Defunciones Generales Proyecciones Poblacionales 2010-2020

Unidades:

Defunciones de personas por homicidios cada 100.000 habitantes

Interpretación:

Por cada 100.000 habitantes, un total de X personas fallecieron a causa de homicidios en el periodo t

Fórmula

\[ TMH^t= \frac{\sum_i^n{x_{i}^t}}{N^t} * 100.000 \]

  • \(TMH^t\) : Tasa de homicidios por cada 100.000 habitantes en el periodo/año t.

  • \(x_{i}^t\) : Defunción del individuo i en el periodo t por a causa de homicidio.

  • \(i\) : Defunción por homicidio individuo i.

  • \(N\) : Proyección poblacional en el periodo t.

  • \(k\) : Constante=100.000

NOTA:

La agrupación de las causas de muertes por homicidio se realiza en función de la Clasificación Estadística Internacional de Enfermedades y Problemas Relacionados con la Salud (CIE):

  • Homicidios: CIE-10 (X85-Y09)

Para más detalle ver Clasificador CIE-10

fc_homicidios <- function(data, 
                          var) { #var: variable de muertes violentas mor_viol

  var <- enquo(var)

  data %>% 

    mutate(homicidio= ifelse(str_detect(string = !!var, 
                                              pattern = '^HOMICI'), 
                                   yes = T, 
                                   no = F)
    )
}

fgrupos <- 

  function(data, vars=c(), varindic=NULL) {

    if(is.null(varindic)){

    data %>% group_by_at(.vars = vars) %>% count() %>% ungroup()

    } else {

    data %>% filter_at(.vars = varindic, .vars_predicate =  all_vars(.==1)) %>% 

      group_by_at(.vars = c(vars)) %>% count() %>% ungroup()

    }

  }

#f3. funcion que agrega los datos en funcion de las variables de agrupacion 

summarizador <- function(tabla, vars=NULL, varindicad=NULL) {

  varst <- c(varindicad, vars, 'anio_bs')


  t <- tabla  %>% group_by_at(.vars = varst) %>% 

    summarise(casos=sum(n, na.rm = T)) %>% ungroup() 

  if(!is.null(varindicad)){

    t %>% 

      filter_at(.vars = varindicad, .vars_predicate = all_vars(!is.na(.)))

  }

  t
}

f_t_homicidios <- function(tabla,
                           proyecciones=proy_prov, 
                           vars=NULL, 
                           varindicad=NULL) {

t <- 

  tabla %>% 

  summarizador(vars = vars,
               varindicad = NULL)

den <- 

  proyecciones %>%

  summarizador(vars = vars,
               varindicad = NULL)

tf <- 

  t %>% 

  left_join(den,

            by = c('anio_bs',vars)

  ) %>% 

  rename(numerador=casos.x,
         denominador=casos.y) %>% 

  mutate(indicador=numerador/denominador*100000)

tf

}
